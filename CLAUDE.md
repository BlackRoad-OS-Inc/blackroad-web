# CLAUDE.md — blackroad-web

Next.js 15 frontend dashboard for BlackRoad OS.

## Stack
- Next.js 15 / React 19 / TypeScript 5
- Tailwind CSS 4 (via PostCSS)
- Zustand (state management)
- Vitest (testing)

## Commands
`npm run dev` — dev server, `npm run build` — production build, `npm run typecheck` — check types, `npm test` — tests

## Brand System
Colors: Hot Pink #FF1D6C, Amber #F5A623, Violet #9C27B0, Electric Blue #2979FF
Spacing: Golden ratio (8, 13, 21, 34, 55, 89, 144px)
